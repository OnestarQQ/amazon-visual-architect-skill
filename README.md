# Amazon Visual Architect Skill

> 🎨 亚马逊全案视觉架构师 AI 技能包 v8.2

## 📖 简介

Amazon Visual Architect 是一个专业的电商视觉设计 AI 技能，能够智能分析产品特性，提取统一品牌基因，生成高质量的 Amazon Listing 主图和 A+ 页面设计 Prompt。

### 🌟 核心价值

- **🎯 专业性**: 基于真实电商设计经验，15种专业布局模板
- **🎨 一致性**: 统一品牌视觉基因，确保设计风格协调
- **🚀 效率**: 批量生成专业 Prompt，大幅提升设计效率  
- **🌍 本土化**: 支持多区域文化适配和语言本土化
- **🛡️ 合规性**: 内置侵权词过滤，确保平台合规
- **🎨 平台适配**: 针对主流AI绘图平台优化，即拷即用

## 🎨 功能特性

### 智能品牌基因提取

- **🎨 统一视觉主题**: 自动提取品牌主色、背景策略、文化适配
- **🛡️ 字体系统锁定**: 5大字体分类，智能选择最适合字体
- **🎲 图标系统**: 实心圆/描边圆/实心方块三种风格

### 智能分流策略

- **📦 Listing 主图**: 功能导向，L1-L7 专业布局
- **🎬 A+ 图片**: 品牌导向，A1-A9 营销布局  
- **🔀 混合模式**: 50:50 智能分配，确保全案完整

### 15种专业布局模板

#### Listing 布局 (L1-L7)
- **L1 文本叠加面板**: 功能说明+视觉面板
- **L2 卖点特征块**: 场景图+卖点模块阵列
- **L3 极简主义**: 大场景+醒目标题
- **L4 元素衍生物**: 产品形态衍生创意布局
- **L5 圆形特写**: 主图+圆形细节特写
- **L6 结构分解**: 3D层次结构分析
- **L7 场景搭配**: Flat Lay 俯拍风格

#### A+ 布局 (A1-A9)  
- **A1 悬浮图文**: 沉浸场景+半透明面板
- **A2 对角切片**: 斜线分割多功能展示
- **A3 卡片阵列**: 标准化信息矩阵
- **A4 景深留白**: 极简摄影+自然构图
- **A5 结构拼图**: Bento 风格非对称布局
- **A6 流程进化**: 线性叙事+品牌植入
- **A7 辐射分析**: 核心产品+四周信息块
- **A8 沉浸全景**: 电影级构图+强景深
- **A9 多视窗**: 场景分解+产品特写

### 多维度本土化

- **🌍 区域适配**: 美国/欧洲/亚洲/中东文化风格
- **👥 人群定位**: 年龄/性别/职业/兴趣精准匹配
- **🗣️ 语言支持**: 中英日等多语言文字生成

## 🔧 参数配置

### 必需参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `customer_keywords` | Array | 核心卖点列表 | `["防水", "轻便", "大容量"]` |
| `salesRegion` | String | 销售区域 | `"美国"`, `"欧洲"`, `"中国"` |
| `目标受众` | String | 用户画像 | `"25-35岁职场女性"` |
| `platform` | String | 发布平台 | `"Amazon"`, `"TikTok"` |
| `outputNum` | Number | 输出数量 | `6`, `8`, `12` |

### 可选参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `GenerateAPlus` | Boolean | `true` | 是否生成A+图片 |
| `language` | String | `"英语"` | 图片文字语言 |
| `ListingUserSelectedSize` | String | `"--ar 1:1"` | 主图尺寸比例 |
| `UserSelectedSize` | String | `"--ar 1464:610"` | A+图尺寸比例 |
| `brandKey` | Object | `null` | 自定义品牌基因 |
| `我的侵权词` | Array | `[]` | 禁用词列表 |

### brandKey 结构

```json
{
  "brandColor": "#2196F3",          // 品牌主色
  "fontStyle": "几何无衬线体",        // 字体风格  
  "iconStyle": "实心圆",             // 图标样式
  "customSetting": "额外定制要求"     // 自定义设置
}
```

## ⚠️ 注意事项

### 使用限制

- **图片来源**: 严禁复制用户上传图片场景，必须自行设计
- **颜色规范**: 非黑白灰产品禁用黑白灰品牌色
- **内容唯一性**: 每张图的 selling_point 必须唯一
- **文字合规**: 不能包含色值代码和设计术语

### 最佳实践

1. **📸 图片准备**: 提供白底图+实拍图，确保产品信息完整
2. **🎯 卖点规划**: 3-6个差异化卖点，覆盖功能/品质/体验
3. **👥 受众描述**: 具体年龄段+职业+兴趣，避免泛泛而谈
4. **🌍 区域考虑**: 充分了解目标市场文化偏好和审美习惯
5. **🔄 迭代优化**: 根据转化数据持续优化视觉策略
